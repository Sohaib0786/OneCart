# 🛒 OneCart – AI Driven Shopping Platform

OneCart is a modern AI-powered eCommerce shopping platform designed to deliver a personalized, fast, and intelligent shopping experience. 
It leverages AI-driven recommendations, a scalable MERN stack architecture , and secure JWT-based authentication to provide a production-ready online shopping solution.

---

##  Key Features

###  AI-Powered Features

* Personalized product recommendations based on user behavior
* Smart product search with relevance ranking
* AI-assisted product discovery
* Intelligent suggestions for similar and trending products

###  User Features

* User registration & login (JWT Authentication)
* Secure authorization & protected routes
* Browse products by category
* Add to cart & manage cart
* Wishlist functionality
* Order placement & order history

###  Admin Features

* Admin authentication & role-based access
* Product management (Add / Update / Delete)
* Category management
* Order management dashboard
* User management

###  Payments (Optional / Extendable)

* Cash on Delivery (COD)
* Online payments integration ( Razorpay – extendable)

### 📦 Order Management

* Real-time order status updates
* Invoice generation
* Order history tracking

---

## 🧱 Tech Stack

### Frontend

* React.js
* React Router
* Context API / Redux (State Management)
* Axios
* Tailwind CSS / CSS Modules

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication
* RESTful APIs


## 🔐 Authentication & Authorization

* JWT-based authentication
* Secure password hashing using bcrypt
* Role-based authorization (User / Admin)
* Protected routes for sensitive operations

---

## 🧠 AI Recommendation Workflow

1. User interaction data (views, cart, purchases) is collected
2. AI engine processes behavior patterns
3. Personalized recommendations are generated
4. Results are served via backend APIs
